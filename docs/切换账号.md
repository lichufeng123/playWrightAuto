npx playwright test --ui 跑的是 playwright.config.ts 里解析出来的环境：

默认环境：test（因为 PW_ENV 不设置时默认为 test），所以默认是测试环境 + testUser。
切正式环境（二选一）：
PowerShell：$env:PW_ENV='prod'; npx playwright test --ui
npm 脚本：npm run ui:prod
切账号：
PW_USER：$env:PW_USER='prodUser' / testUser（不填会按 baseURL 自动选默认账号）
临时换号：$env:LOGIN_PHONE='xxx'; $env:LOGIN_CODE='1234'
